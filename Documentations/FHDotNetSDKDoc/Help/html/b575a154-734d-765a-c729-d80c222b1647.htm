Namespaces
NamespaceDescription
FHSDK
The parent name space defined by FeedHenry .Net SDK. It is defined inside the FHSDK.dll assembly, which is a Portable Class Library. The FHSDK.dll assembly can be referenced by other PCL projects.
FHSDK.API
Contains implementations for accessing FeedHenry APIs. Defined in the FHSDK.dll.
FHSDK.Droid
Contains the entry class of the FeedHenry Xamarin SDK for Android platform. It's defined in the FHXamarinAndroidSDK.dll. To use the FeedHenry SDK, both FHSDK.dll and FHXamarinAndroidSDK.dll should be referenced by your Xamarain Android project, and initialise the SDK using the FHClient class in this name space.
FHSDK.FHHttpClient
Contains implementation of a HttpClient used by the FeedHenry .Net SDK. Defined in the FHSDK.dll assembly.
FHSDK.Phone
Contains the entry class of the FeedHenry SDK for Windows Phone 8 platform. It's defined in the FHSDKPhone.dll. To use the FeedHenry SDK, both FHSDK.dll and FHSDKPhone.dll should be referenced by your WP8 project, and initialise the SDK using the FHClient class in this name space.
FHSDK.Services
Contains implementations of a few services used by the FeedHenry .Net SDK. The interfaces are defined in the FHSDK.dll assemably, then each platform's assembly contains platform-specific implementations of these services.
FHSDK.Touch
Contains the entry class of the FeedHenry Xamarin SDK for iOS platform. It's defined in the FHXamarinIOSSDK.dll. To use the FeedHenry SDK, both FHSDK.dll and FHXamarinIOSSDK.dll should be referenced by your Xamarain iOS project, and initialise the SDK using the FHClient class in this name space.
